# E-Commerce-Sales-Intelligence-Dashboard
Interactive Power BI dashboard analyzing e-commerce sales, profitability, customer behavior, and seasonal trends using SQL, Excel, Python, and Power BI.
E-Commerce Sales Intelligence Dashboard

This project analyzes an e-commerce Superstore dataset and converts raw sales data into meaningful business insights using SQL, Python, Excel and Power BI.

The objective of this project is to simulate a real business scenario where a company wants to understand sales performance, profitability, customer behavior and product trends through data analytics.

Project Overview

Businesses generate large amounts of sales data, but raw data has no value without analysis.
This project follows the complete data analyst workflow:

Data → Cleaning → Analysis → SQL Queries → Dashboard → Insights

The final output is an interactive Power BI dashboard designed to help decision-makers understand:

Sales performance
Profit trends
Customer purchasing behavior
Product and regional performance
Tools and Technologies Used
Python (Pandas, NumPy) for data cleaning and preprocessing
SQL for business queries and analysis
Excel for initial data exploration and validation
Power BI for data visualization and dashboard creation
Project Workflow
1. Data Cleaning (Python)
Removed null values and duplicates
Standardized column formats
Prepared dataset for analysis
2. SQL Analysis

Key business questions answered:

Total sales and profit
Sales by category and region
Monthly sales trends
Top selling products
Customer segment performance
3. Dashboard Creation (Power BI)

A two-page interactive dashboard was created to present business insights clearly.

Dashboard Features
 Sales Overview
Total Sales, Profit and Orders KPIs
Monthly sales trend
Sales by category
Sales by region
Segment performance
 Product and Profit Analysis
Top 10 products by sales
Profit by category
Discount vs profit analysis
Customer segment profitability
Key Insights
The Technology category generates the highest revenue
Higher discounts negatively impact profitability
The West region shows the strongest sales performance
A small number of products generate the majority of revenue (Pareto principle)
